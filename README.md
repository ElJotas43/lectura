# Visualización de Datos X-Y con Flask

## Nombre: JuanPablo Pinto

## Descripción

Aplicación desarrollada con Flask que recibe coordenadas X-Y desde una APK y las representa mediante distintas visualizaciones.

## Vistas

- Cuadrantes
- Radar 8x8
- Mapa de Posición

## Tecnologías

- Python
- Flask
- HTML
- CSS
- Jinja
